Abstract

1. Definition

Φ has units of seconds. E is local energy density; ρ₀ is a reference mass density.

2. Spatial Structure

∇Φ = (∇E) / (ρ₀ c³) ∇²Φ = (∇²E) / (ρ₀ c³)

Gradient → tilt. Laplacian → curvature.

3. Evolution (Toy PDE)

∂²Φ/∂t² = c_eff² ∇²Φ + S(x,t)

A simple wave equation for Φ with adjustable propagation speed c_eff.

4. Dimensional Checks

Φ is dimensionally consistent; replaces all use of E/c³ as “time.”
